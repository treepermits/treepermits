How to appeal a tree removal decision?

Would you like to challenge a decision to remove trees on one of the properties listed on this site? Take a look at the deadline to file an appeal — if it still hasn't passed, you can do it!

If you need help, join our WhatsApp chat — we can help you file the appeal.
1
Figure out who can file the appeal
  • Are you the next-door neighbor? Great — you'll pay a reduced fee to file.
  • Not next-door? Can you find the next-door neighbor who'd like to file and appear at City Hall when called?
  • You live within 500ft and there's an HOA? The HOA can file for a reduced fee.
  • No HOA but within 500ft? You can still apply, but you'll pay an extra $200.
  • None of the above? Text us in the chat — we might be able to help you submit the appeal without the extra $200.
2
Request the invoice by email
Locate the Decision Application Number (looks like BD25-000000-001) and the address from the table above. Then send an email to PZHearingBoards@miami.gov as soon as possible:
If you are NOT the next-door neighbor:
My name is (NAME) and I am requesting the invoice for the appeal of the Intended Decision Application No. (NUMBER). The subject property address is (ADDRESS).

If you ARE the next-door neighbor:
My name is (NAME) and I am requesting the invoice for the appeal of the Intended Decision Application No. (NUMBER) as an ABUTTING PROPERTY OWNER. The subject property address is (ADDRESS).
You will receive the invoice the same day.
3
Pay the fees
There is a number on the invoice you received. Go to miami.gov/make-a-payment, enter your invoice number, and pay the fees.

Save the receipt! You will need to submit it later.
4
Download the screenshot of radius and the list of residents within 500ft
▶ Show step-by-step instructions
  1. Use this link: https://gis.miami.gov/miamizoning/main
    Type the address of the property in.
  2. On the right side choose this icon and type "500" feet in the pop up "Buffer" window.
  3. Take a screenshot of the radius. You will need to upload it with your appeal. Like this:

    When you save the screenshot, name the file "[TODAY'S DATE] – [property address] – 500ft radius".
  4. Use another icon to download the excel file with mailing addresses of people within the 500ft radius.
  5. Copy appropriate columns into this template:
    Owner NameMailing Street AddressCityStateZipCountryFolio #
           
           
    Do not add, re-arrange, or delete any columns. Use only the seven (7) columns shown: 1) Owner Name 2) Mailing Address 3) City 4) State 5) Zip Code 6) Country 7) Folio Number.
  6. Make sure there are no duplicates, no empty lines. Make sure one owner does not have several mailing addresses (only keep one). Also, if you see a long list of addresses with apartment numbers, you must find out the name of the condominium building or co-op, delete them all and replace with ONLY one line for Condominium Association and one mailing address.

    The number of lines will determine the final price of your appeal, because you are responsible for paying to notify them all by mail about your appeal.
  7. Save the file in the excel (!) format with date in the name: "[TODAY'S DATE] – [property address] – list of owners 500ft radius"
5
Complete the property owner affirmation form
Print this form. Write in pen the number of "property owners shown in the attached Excel file" (the count from your list). Sign, scan, and save with the date and property address in the filename.

7
Submit the appeal online
Submit using this online form. The first page asks for the date — use the date the intended decision was posted.

Upload all of the following:
  • Receipt of paid invoice
  • Screenshot of 500ft radius
  • Excel file with the list of owners within 500ft
  • Scan of the affirmation form (stating how many owners)
  • Scan of your signed appeal letter
  • Any other supporting files (optional): photos of trees, nests, site plans, arborist reports, anything you deem relevant
That's it! Await the city's confirmation that your submission is timely and complete. 🌳
